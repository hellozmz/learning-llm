# 用户先验知识基线

用户已具备：
- Python/numpy/pandas 熟练使用
- 会用 ollama 等工具拉取和运行开源模型
- 能调用 OpenAI 兼容 API 进行简单对话
- 听说过 Transformer、Attention、RAG、LoRA 等概念，但无法解释其原理
- 深度学习零基础：不知道反向传播、梯度下降的数学细节

这确定了教学的起点：**从深度学习基础开始，而非直接从 Transformer 切入**。先建立神经网络的基本心智模型（前向传播、损失函数、梯度下降），再自然过渡到 Transformer。

Implications:
- 前几节课应聚焦 DL 基础，而不是直接跳入 Transformer
- 每节课需要代码 + 理论并重，因为用户是实践型学习者
- 术语保留英文，解释用中文
