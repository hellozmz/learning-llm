Transformer 完整计算图

原始 Transformer (Encoder-Decoder)，从输入到输出，每步的公式、张量形状、计算复杂度
假定：batch=1, seq_len=n, d_model=d, h 个头, dₖ=dᵥ=d/h, FFN 扩展 4×

全局参数

参数形状说明
Token Embedding(vocab_size, d)词表嵌入矩阵
Position Embedding / Encoding(max_len, d)位置编码（可学习或 sinusoid）
W_Q, W_K, W_V（每层每头）(d, dₖ) / (d, dₖ) / (d, dᵥ)Attention 投影
W_O（每层）(hdᵥ, d) = (d, d)Multi-head 输出投影
W₁, b₁（每层 FFN）(d, 4d), (4d,)FFN 升维
W₂, b₂（每层 FFN）(4d, d), (d,)FFN 降维
γ, β × N（LayerNorm）(d,) 各一个每个子层前后各一个 LN

ENCODER

E1 Input Embedding + Positional Encoding

x = Embedding(token_ids) + PositionEncoding(positions)
输入: (n,)         → token IDs
输出: (n, d)       → 每个 token 的 d 维向量
计算量: O(n·d)     → 查表 + 加法，忽略不计

E2 Multi-Head Self-Attention（每层，Post-Norm 原始版）

# 分头投影: 每个 head 独立做
Q = x @ W_Q           # (n, d) @ (d, dₖ) → (n, dₖ)   × h 个头
K = x @ W_K           # (n, d) @ (d, dₖ) → (n, dₖ)
V = x @ W_V           # (n, d) @ (d, dᵥ) → (n, dᵥ)
计算量: 3h × n·d·dₖ ≈ 3n·d²  (dₖ = d/h)

# Scaled Dot-Product Attention（每个头）
S = Q @ Kᵀ            # (n, dₖ) @ (dₖ, n) → (n, n)
计算量: n²·dₖ = n²·d/h

A = softmax(S / √dₖ)  # (n, n) → (n, n)
计算量: O(n²)         ← softmax 逐行

O = A @ V             # (n, n) @ (n, dᵥ) → (n, dᵥ)
计算量: n²·dᵥ = n²·d/h

# 合并多头
Concat = [O₁, O₂, ..., O_h]  # h 个 (n, dᵥ) → (n, h·dᵥ) = (n, d)

# 输出投影
out = Concat @ W_O    # (n, d) @ (d, d) → (n, d)
计算量: n·d²

# 残差连接 + LayerNorm（Post-Norm）
x = LayerNorm(x + Dropout(out))

Attention 总计算量: O(n·d² + n²·d)   ← 3n·d²(投影) + 2n²·d(点积+加权) + n·d²(W_O)

E3 Feed-Forward Network（每层）

h = x @ W₁ + b₁      # (n, d) @ (d, 4d) → (n, 4d)
计算量: 4n·d²

h = ReLU(h)           # (n, 4d) → (n, 4d)
计算量: O(n·d)

out = h @ W₂ + b₂    # (n, 4d) @ (4d, d) → (n, d)
计算量: 4n·d²

# 残差连接 + LayerNorm
x = LayerNorm(x + Dropout(out))

FFN 总计算量: O(8n·d²)   ← 两个线性层各 4n·d²

E4 堆叠 N 层 Encoder

Encoder 每层: O(n·d² + n²·d + 8n·d²) ≈ O(n·d² + n²·d)
N 层 Encoder: O(N·n·d² + N·n²·d)

输出: encoder_output, shape: (n, d)
      这个输出会作为 Decoder Cross-Attention 的 K 和 V

DECODER

D1 Output Embedding + Positional Encoding

y = Embedding(output_token_ids) + PositionEncoding(positions)
输入: (m,)         → 已生成/训练中的目标序列
输出: (m, d)
计算量: O(m·d)

D2 Masked Multi-Head Self-Attention（每层）

与 E2 完全相同，但加了 Causal Mask:

S = Q @ Kᵀ / √dₖ            # (m, dₖ) @ (dₖ, m) → (m, m)
S[i, j] = -∞  if j > i      # 禁止位置 i 看到位置 j（j 在未来）
A = softmax(S)               # (m, m), 上三角 ≈ 0
O = A @ V                    # (m, dᵥ)

计算量: O(m·d² + m²·d)   （与 E2 相同，n→m）

D3 Cross-Attention（每层）

Q = y @ W_Q_dec       # (m, d) @ (d, dₖ) → (m, dₖ)
                        ↑ Q 来自 Decoder 当前 hidden state

K = enc_out @ W_K_dec  # (n, d) @ (d, dₖ) → (n, dₖ)
V = enc_out @ W_V_dec  # (n, d) @ (d, dᵥ) → (n, dᵥ)
                        ↑ K, V 来自 Encoder 输出（E4）

S = Q @ Kᵀ             # (m, dₖ) @ (dₖ, n) → (m, n)
                        ↑ Decoder 的 m 个位置 × Encoder 的 n 个位置
A = softmax(S / √dₖ)   # (m, n)
O = A @ V              # (m, n) @ (n, dᵥ) → (m, dᵥ)

计算量: O(m·d² + n·d² + m·n·d)
        ↑Q投影    ↑K,V投影  ↑Cross-Attn 点积和加权
        注意: m≠n 时计算量和 Self-Attn 不同

D4 Feed-Forward Network + 堆叠 + 输出

FFN: 与 E3 相同，n→m
Decoder 每层: O(m·d² + m²·d + n·d² + m·n·d + 8m·d²)
N 层 Decoder: N × 上述

# 最终输出
logits = decoder_output @ W_embed.T   # (m, d) @ (d, vocab_size) → (m, vocab_size)
probs  = softmax(logits)               # (m, vocab_size)
loss   = CrossEntropy(probs, target)   # 训练时

计算量: O(m·d·vocab_size) ≈ O(m·d·V)  ← V=vocab_size 通常很大(50K+)

计算量总结

操作计算量瓶颈条件
EmbeddingO(n·d)忽略不计
Q/K/V 投影O(n·d²)d 大时（大模型）
Q·Kᵀ（Self-Attn）O(n²·d)n 大时（长上下文）
A·V 加权O(n²·d)n 大时
W_O 投影O(n·d²)d 大时
FFNO(8n·d²)d 大时
Cross-Attn Q·KᵀO(m·n·d)m,n 都大时
LM HeadO(m·d·V)V=50K+ 时不可忽略

关键分界线：

  • 短序列 (n < d): O(n·d²) 主导 → FFN 和投影是瓶颈
  • 长序列 (n ≫ d): O(n²·d) 主导 → Attention 是瓶颈，二次增长碾压一切
  • 以 GPT-2 Small 为例 (d=768): 当 n > 768×(8/1) ≈ 6K tokens 时，Attention 开始超过 FFN 成为瓶颈
