第1课：什么是神经网络？

从你熟悉的线性回归，到"学习"的本质
预计时间：25 分钟  |  2026-06-17

1. 你其实已经会了：线性模型

做量化的你一定不陌生——用过去 N 天的价格预测明天的价格：

price_tomorrow = w₁·price_today + w₂·price_yesterday + ... + b

这就是一个线性模型。它有：

  • 输入 x（特征，比如今日价格、成交量）
  • 权重 w（每个特征的重要性）
  • 偏置 b（基础值，不依赖任何特征）
  • 输出 ŷ（预测值）

写成数学形式：ŷ = w₁x₁ + w₂x₂ + ... + wₙxₙ + b

这本质上就是一个神经元。神经网络的最小单元和线性回归没有本质区别。

输入 x₁ ──→ (× w₁) ┐ ├──→ (+) ──→ 输出 ŷ 输入 x₂ ──→ (× w₂) ┘ ↑ 偏置 b
图1：一个神经元 = 加权求和 + 偏置
关键洞察：神经网络并不是什么神秘的东西。一个神经元就是对输入的加权求和。一千个神经元就是一千个加权求和，只是组织方式和训练方法不同。

2. 为什么线性不够？—— 激活函数

假设你用两层线性模型：

h = W₁x + b₁        （第一层）
ŷ = W₂h + b₂        （第二层）
→ ŷ = W₂(W₁x + b₁) + b₂ = (W₂W₁)x + (W₂b₁ + b₂)
→ 这仍然只是一个线性模型！

多层线性变换叠在一起还是线性的。 这意味着无论堆多少层，模型只能画直线，永远画不出曲线。

解决方案：在每层之间插入一个非线性函数——激活函数（activation function）。

h = ReLU(W₁x + b₁)   ← ReLU 就是那个非线性
ŷ = W₂h + b₂

最常用的激活函数：

ReLU(x) = max(0, x) GELU(x) ≈ x·Φ(x) │ │ │ ╱ │ ╱ │ ╱ │ ╱ │ ╱ │ ╱ │ ╱ │ ╱ (平滑曲线) ───┼●───── ─────●───── │ │

ReLU（Rectified Linear Unit）是目前最常用的激活函数：负数全变 0，正数保持不变。简单、计算快、效果好。现代 LLM 也多用 ReLU 的变体（如 SwiGLU）。

一句话总结：线性变换 + 非线性激活 = 万能函数逼近器。这就是神经网络的表达能力来源。

3. "学习"到底是什么意思？—— 梯度下降直觉

模型一开始的参数（w, b）是随机的，预测当然不准。学习 = 不断调参让预测更准。

这个过程分三步：

Step 1: 算误差（前向传播）

prediction = model(x, w)       # 用当前参数算预测值
loss = (prediction - y_true)²   # 均方误差：预测和真实差多少

Step 2: 算梯度（反向传播）

梯度 = "如果我把 w 调大一点点，loss 会怎么变？"

对每个参数都算这个偏导数。PyTorch 等框架自动帮你算（autograd）。

Step 3: 更新参数（梯度下降）

w_new = w_old - learning_rate × gradient

沿着使 loss 下降的方向挪一小步。学习率控制步长——太大跳过最优解，太小学得太慢。

类比：你蒙着眼睛在山上走，想走到山谷最低点。每一步你探一下脚下的坡度（梯度），然后往最陡的下坡方向走一小步（梯度下降）。重复足够多次，你就能走到谷底。

4. 用代码看一眼

不需要装任何东西，就在脑子里跑一遍这段伪代码：

# 一个最简单的"学习"循环
w = 0.5          # 初始权重（瞎猜的）
b = 0.0          # 初始偏置
lr = 0.01        # 学习率

for step in range(100):
    # 前向传播：算预测
    y_pred = w * x + b

    # 算 loss
    loss = ((y_pred - y_true) ** 2).mean()

    # 反向传播：算梯度（PyTorch 帮你做）
    grad_w = 2 * ((y_pred - y_true) * x).mean()
    grad_b = 2 * (y_pred - y_true).mean()

    # 梯度下降：更新参数
    w = w - lr * grad_w
    b = b - lr * grad_b

运行 100 步后，w 和 b 会自动收敛到让 loss 最小的值——这就是"学习"。
